# etcd-hth-lab

## 项目简介

etcd-hth-lab 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单一域名或具体网站，而是作为一个通用的页面存放与展示场所，方便对各类静态页面进行版本管理、组织与访问。

## 目录说明

```
etcd-hth-lab/
├── docs/          # 存放所有 HTML 页面及其相关资源（如 CSS、JS、图片等）
├── archive/       # 历史版本或不再活跃的页面归档
└── README.md      # 本文件
```

- **docs/**：主要页面目录，每个子文件夹或文件代表一个独立页面。
- **archive/**：用于存放已过时或暂不维护的页面，便于查阅历史内容。

## 页面归档说明

本仓库中的每个 HTML 页面均为独立单元，可单独访问，彼此之间无强制关联。页面内容可能随时间更新或替换，旧版本将被移入 `archive/` 目录以保持主目录的整洁。

若需引用或分发本仓库中的页面，请直接使用对应的原始文件路径或 GitHub Pages 提供的访问地址（若已启用 Pages 服务）。

## 维护说明

- 页面更新时，请确保在 `docs/` 下保留当前版本，并将旧版本移动至 `archive/`。
- 新增页面请遵循现有目录结构，避免文件命名冲突。
- 不建议在本仓库中存放动态内容或依赖外部服务的脚本，以保持页面的独立性和可归档性。
- 如有疑问或希望贡献页面，请提交 Issue 或 Pull Request。

## 许可

本仓库中的页面内容遵循各自页面的原始许可。若无特别说明，默认采用 [MIT 许可证](LICENSE)。
